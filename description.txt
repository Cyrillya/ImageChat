A mod that allows you to send images to chat or take in-game screenshots from within tModLoader.
When you copied an image, pressing Ctrl+V when chat is open will send the image to chat.
You need to bind a key to take in-game screenshots. The image will be immediately sent to chat and will be copied to the clipboard.
Sending images in multiplayer games may contain a lot of data, make sure you have enough traffic.
Currently you can only send images on Windows, MacOS and Linux are not yet supported
Open source on GitHub: https://github.com/Cyrillya/ImageChat
Report any bugs in discord server: https://discord.gg/rEmGMQv5z7
